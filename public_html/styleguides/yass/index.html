    Plain HTML Elements

    このモジュールパッケージは、HTMLドキュメントを構成するための最も基礎となるシンプルなモジュールを提供します。

    HTML Elements

    HTML

    HTMLモジュールは、入力されたHTMLコードをそのままソースとして出力します。

    < や " などのHTML特殊文字を画面に反映したい場合は、&lt; &quot; のように入力してください。

    CODE: module template code

    Plain Text

    Plain Text モジュールは、入力されたテキストを、HTMLに変換してそのまま画面へ表示します。

    < や " などのHTML特殊文字変換が適用され、改行は <br /> に変換されます。

    CODE: module template code

    Markdown

    Markdown モジュールは、Markdown形式のテキストを、HTMLに変換して出力します。

    Markdown形式は、簡易な文法で構造化されたHTMLドキュメントを記述できるファイル形式です。たとえば、行の先頭を ## で書き始めたら <h2> に変換される、- で書き始めたら <ul> と <li> によるリスト表現に変換されるなど、さまざまな表現方法が規定されています。

    その他の表現方法などについて詳しくは、Markdown のドキュメントなどを参考にしてください。

    CODE: module template code

    画像要素を入力します。

    リンク先を入力して、画像リンクを作成することもできます。
